# PokeQuest

Welcome to PokeQuest! This is a gamified task management application designed to make your daily to-do list more engaging and fun.

In PokeQuest, you can:
- **Create and complete daily tasks** to earn experience points (XP).
- **Level up** your character as you become more productive.
- **Maintain a daily streak** to earn XP multipliers.
- **Compete against rivals** who also gain XP throughout the day.
- **Unlock achievements** for reaching milestones.
- **Receive notifications** about your progress and rival activity.

This project is built with Next.js, React, Tailwind CSS, and uses Genkit for its AI-powered features. The backend is powered by Firebase Firestore for real-time data synchronization.

To get started, explore the application and begin completing tasks to see your character grow!
